## 使用方式

1. 开发新模块前，阅读对应的前端/后端模板
2. 参照模板中的代码骨架和规范编写
3. 按照 Checklist 逐项完成，不遗漏
4. 如果发现模板与现状不符，更新模板（模板服务于项目）
